# Mybrary
> 소장 도서로 사람을 잇고, 도서의 가치를 발견하다.

</br>

## Mybrary가 추구하는 가치
- 사람과 사람을 도서로 잇고, 도서로 연결한다.
- 각자의 도서 가치를 나누고 공유하며, 그 속에서 더 큰 가치를 발견한다.
